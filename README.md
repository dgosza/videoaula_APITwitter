## Como baixar e iniciar o projeto?

``` bash 

#Baixando o repositorio
git clone git@github.com:dieezs/videoaula_APITwitter.git

#Entrando na pasta do projeto
cd videoaula_APITwitter

#Instalar as dependencias do projeto
npm install

#Iniciar o projeto
npm start

```
